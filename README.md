# 🌊 Flood Impact on Agriculture in Assam (Using Google Earth Engine)

## 📌 Project Overview
This project analyzes the **impact of floods on agricultural land in Assam, India** using satellite data and **Google Earth Engine (GEE)**.  
The study focuses on identifying flooded areas and understanding their effect on vegetation health using **NDVI**.

---

## 🎯 Objectives
- Analyze flood extent in Assam
- Study vegetation condition before and after floods
- Understand flood impact on agricultural regions
- Perform year-wise comparison

---

## 🛰️ Data Used
- **Sentinel-2** (Optical satellite data)
- **MODIS** (Vegetation indices)
- Administrative boundary data for Assam

---

## 🛠️ Tools & Technologies
- Google Earth Engine (JavaScript API)
- Remote Sensing & GIS concepts
- NDVI (Normalized Difference Vegetation Index)

---

## 📊 Methodology
1. Define Assam boundary
2. Filter satellite images by date & cloud cover
3. Compute NDVI
4. Identify flooded areas
5. Compare vegetation before and after floods

---

## 📌 Current Status
✔ Assam NDVI mapping  
✔ Flood layer visualization  
🔄 District-wise analysis (in progress)  
🔄 Agriculture impact quantification (in progress)

---

## 📊 Results
- NDVI maps show significant vegetation reduction in flooded areas
- Flood impact on agricultural land varies across districts
- Post-flood recovery patterns were observed using time-series data
